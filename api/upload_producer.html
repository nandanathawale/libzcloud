Upload Producer¶

class UploadProducer¶
read(buffer, bytes, error)¶
Parameters:
  • buffer – storage for the read bytes (large enough to hold the requested number of bytes)
  • bytes – the requested number of bytes to read
  • error – optional storage for an error
Returns:

the number of bytes actually read and copied into buffer
reset(error)¶

Reset the producer. The next call to read should return the same data as the first one.

Parameters:
  • error – optional storage for an error
Returns:

true on success and false on error
get_size(error)¶
Parameters:
  • error – storage for an error
Returns:

the number of bytes that this object will produce
calculate_md5(error)¶
Parameters:
  • error – optional storage for an error
Returns:

the MD5 hash of the data that this object will produce, or null on error
